# Recorded video fixture

`pedestrians.mp4` is a four-second, 320×240, 5 FPS H.264 excerpt derived from
OpenCV's `samples/data/vtest.avi` fixture. It is bundled so simulator validation
does not depend on camera hardware or network video playback.

Source: <https://github.com/opencv/opencv/blob/master/samples/data/vtest.avi>
